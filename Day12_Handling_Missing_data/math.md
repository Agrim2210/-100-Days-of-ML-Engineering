# Conceptual Foundations — Missing Data

Missing data handling involves replacing or removing
incomplete values in a dataset.

Common strategies include:

Mean Imputation:
Replacing missing values with the average of the column.

Median Imputation:
Replacing missing values with the median value,
which is more robust to outliers.

Mode Imputation:
Used for categorical variables.

There is no single best method; the choice depends on
data distribution and context.
