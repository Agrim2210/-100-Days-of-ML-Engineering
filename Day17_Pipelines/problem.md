# Day 17 — Pipelines and ColumnTransformer

## Problem

Applying preprocessing steps manually before model training can lead to data leakage and inconsistent transformations.

Additionally, real-world datasets contain different types of data that require different preprocessing techniques.

## Objective

This experiment demonstrates the use of Pipelines and ColumnTransformer to build a clean and reliable ML workflow.

The goal is to ensure correct preprocessing and prevent data leakage while handling mixed data types effectively.
