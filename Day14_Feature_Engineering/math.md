# Mathematical Foundations — Feature Engineering

## Interaction Features

Interaction features are created by multiplying
two or more features.

Example:

x_interaction = x1 * x2

This captures relationships between variables.

---

## Polynomial Features

Polynomial features extend features to higher powers.

Example:

x², x³

Polynomial expansion allows linear models to capture
non-linear relationships.

---

## Key Concept

Feature engineering transforms input space into a form
where linear models can perform better.
