# Insights — Classification Metrics

Accuracy alone does not always represent model quality.

In imbalanced datasets a model may achieve high accuracy
by predicting the majority class most of the time.

Precision focuses on prediction reliability,
while recall focuses on detection capability.

F1 Score provides a balanced metric when both precision
and recall are important.

Choosing the correct evaluation metric depends on the
real-world cost of false positives and false negatives.
