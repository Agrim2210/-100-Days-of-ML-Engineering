# Day 7 — Classification Evaluation Metrics

## Problem

In classification problems the target variable represents categories
such as spam vs not spam, fraud vs legitimate transaction,
or disease vs healthy patient.

Using regression metrics like MSE or RMSE is not appropriate for
classification tasks.

Therefore we require evaluation metrics specifically designed for
classification models.

## Objective

This experiment explores fundamental classification evaluation metrics:

• Accuracy  
• Precision  
• Recall  
• F1 Score  

These metrics help us understand model performance in situations
where class imbalance or prediction errors have different costs.
