# Day 11 — Data Leakage

## Problem

Machine learning models sometimes achieve very high
performance during training but fail in real-world usage.

This often occurs due to data leakage, **where the model is exposed to information that would not be available at prediction time.**

## Objective

This experiment demonstrates how data leakage can
artificially inflate model performance.

It also shows how removing leakage leads to a more
realistic evaluation of the model.
