# Key Insights — Day 6 (Regression Metrics)

## Error Measurement

Regression metrics quantify how far predictions are from actual values.

Different metrics highlight different aspects of model performance.

---

## MSE vs MAE

MSE penalizes large errors more strongly because errors are squared.

MAE treats all errors equally and is less sensitive to outliers.

---

## RMSE Interpretation

RMSE provides error values in the same unit as the target variable, making it easier to interpret model performance.

---

## R² Score Meaning

R² measures how well the model explains the variance in the dataset.

Higher R² values indicate better predictive capability.

---

## Practical Insight

Using multiple metrics provides a more complete understanding of regression model performance.
