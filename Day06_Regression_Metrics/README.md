# Day 6 — Regression Evaluation Metrics

## Problem

After training a regression model, we need a way to measure how well it predicts continuous values.

Simply observing predictions is not sufficient.
We need quantitative metrics that measure prediction error.

## Objective

This experiment explores common regression evaluation metrics:

* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* Mean Absolute Error (MAE)
* R² Score

These metrics help evaluate the performance of regression models.
