# Day 1 — Linear Regression From Scratch

## Problem

Build a Linear Regression model **from scratch using NumPy** without using any machine learning libraries.

The goal is to understand:

- Gradient Descent optimization
- Mean Squared Error loss function
- Convergence behavior
- Learning rate sensitivity

## Dataset

A simple regression dataset is used to predict a target variable from input features.

Example use cases of linear regression in real-world systems:

- House price prediction
- Demand forecasting
- Revenue prediction
- Risk estimation

## Objectives

1. Implement Linear Regression using gradient descent.
2. Minimize Mean Squared Error loss.
3. Compare predictions with sklearn implementation.
4. Analyze convergence of the loss function.
