# Mathematical Foundations — Precision Recall Curve

## Precision

Precision = TP / (TP + FP)

This metric measures the fraction of predicted
positive samples that are actually correct.

---

## Recall

Recall = TP / (TP + FN)

This metric measures the fraction of actual
positive samples correctly detected by the model.

---

## Precision Recall Curve

The Precision-Recall curve plots Precision
against Recall for different classification thresholds.

It highlights the trade-off between identifying
more positive samples and maintaining prediction purity.

---

## Average Precision

Average Precision represents the area under
the Precision-Recall curve.

Higher values indicate better model performance,
especially in imbalanced classification problems.
