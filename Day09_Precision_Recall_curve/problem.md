# Day 9 — Precision Recall Curve

## Problem

In classification tasks with imbalanced datasets,
metrics like accuracy or even ROC curves may
fail to represent true model performance.

When the positive class is rare, evaluation
methods must focus more on detecting positive
samples correctly.

## Objective

This experiment explores the Precision-Recall curve,
which visualizes the trade-off between precision
and recall across different classification thresholds.

The experiment also computes Average Precision,
which summarizes the Precision-Recall curve
into a single evaluation score.
