# Day 8 — ROC Curve and AUC

## Problem

Classification models often output probability scores instead
of binary predictions.

Different decision thresholds produce different classification
results, affecting metrics like precision and recall.

Evaluating a model using a single threshold can therefore
provide an incomplete picture of model performance.

## Objective

This experiment explores the Receiver Operating Characteristic
(ROC) curve and the Area Under the Curve (AUC).

These tools allow evaluation of model performance across
all possible classification thresholds.
