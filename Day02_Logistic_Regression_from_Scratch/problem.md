# Day 2 — Logistic Regression From Scratch

## Problem

Build a Logistic Regression classifier from scratch using NumPy.

The goal is to understand how linear models can be adapted for classification tasks by modeling probabilities using the sigmoid function.

## Dataset

The dataset contains user information such as:

- Age
- Salary
- Experience

The task is to predict whether a user purchases a laptop.

Target variable:

Bought_Laptop
0 → No
1 → Yes

## Objectives

1. Implement the sigmoid function.
2. Implement logistic regression using gradient descent.
3. Minimize binary cross-entropy loss.
4. Compare results with sklearn LogisticRegression.
