# Day 10 — Cross Validation

## Problem

Evaluating machine learning models using a single
train-test split can produce unreliable results.

Model performance may vary depending on how the
data is randomly divided into training and testing sets.

## Objective

This experiment explores cross validation, a technique
that evaluates model performance across multiple
data splits.

The goal is to obtain a more reliable estimate of
model performance by averaging results from
multiple training and validation cycles.
