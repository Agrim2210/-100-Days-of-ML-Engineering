## Insights — Cross Validation

Cross validation provides a more reliable estimate
of model performance compared to a single train-test split.

By training and evaluating the model across multiple
data partitions, we reduce the effect of randomness
in data splitting.

Stratified cross validation is particularly useful for
classification tasks where maintaining class
distribution across folds is important.

Using cross validation helps machine learning
practitioners select models and hyperparameters
with greater confidence.
